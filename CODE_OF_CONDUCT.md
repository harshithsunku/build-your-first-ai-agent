# Code of Conduct

## The short version

Be respectful. Assume good faith. Help each other learn.

## The slightly longer version

We follow the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).
The full text is at that link; the gist:

- **Welcoming and inclusive language.** Disagree with the idea, not the person.
- **No harassment**, public or private. This includes trolling, demeaning comments, personal or
  political attacks, doxxing, or unwelcome sexual attention.
- **No off-topic noise** in issues and PRs. Keep discussions focused on the project.
- **Maintainers will moderate.** Comments, commits, code, issues, and other contributions that
  don't follow this code may be edited, hidden, or removed at the maintainers' discretion.

## Reporting

If you experience or witness behavior that violates this code, open a confidential issue or
contact the maintainer directly via GitHub. Reports are taken seriously and handled with
discretion.

## Scope

This code applies in all project spaces — issues, pull requests, discussions, and any other
official channel — and when an individual is representing the project in public spaces.
